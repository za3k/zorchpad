The original ae.c was written by Anthony Howe and released into the public domain.

This version is adapted to work with the zorchpad (use raw keyboard input and video output).

Keys

h/j/k/l cursor left/down/up/right
H/L		word (left/right)
J/K		page (down/up)
[/]		start of line / end of line
t/b		top of file / bottom of file
i		insert (ctrl-L exits)
x		delete character
W		save file
R		redraw
Q		quix

other	movegap
